html_controls_url_addon

May be assigned the URL of a Web page to implement HTML-based controls for the performance. This is normally used only if there is a secondary display to use for these controls, so that the primary display can become fullscreen. The resulting Web page can obtain a reference to this piece from its window.opener, which is the window that opens the HTML controls window, and the opener can be used to control all aspects of the piece.

For this to work the HTML controls and the piece must have the same origin.
