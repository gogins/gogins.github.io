Window Management API demo

This page provides a basic demo of the Window Management API, which in supporting browsers provides a button to open multiple windows to provide an HTML, CSS, and JavaScript learning environment, tailored to whether you have one, two, or more screens available.

On non-supporting browsers it will provide links to the different windows instead, so users of those browsers can have access to that content, and can choose to arrange the opened pages in new tabs or windows as they see fit.
