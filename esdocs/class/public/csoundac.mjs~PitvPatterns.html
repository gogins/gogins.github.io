    PitvPatterns

    Extends:

    statefulpatterns.mjs~StatefulPatterns → PitvPatterns

    Creates a class to hold state and defines Patterns for creating and using that state to work with CsoundAC PITV groups. An instance of this class must be created at module scope and passed to the relevant Patterns.
