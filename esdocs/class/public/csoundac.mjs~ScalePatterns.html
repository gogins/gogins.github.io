    ScalePatterns

    Extends:

    statefulpatterns.mjs~StatefulPatterns → ScalePatterns

    Creates a class to hold state, and defines Patterns for creating and using that state to work with CsoundAC Scales. An instance of this class must be created at module scope and passed to the relevant Patterns. The constructor sets the number of voices in Chords associated with the Scale, by default 4.

    State is co-ordinated with the triggers of the Patterns by only updating the state when the input of the Pattern changes.
